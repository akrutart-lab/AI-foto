# NeuroShot — лендинг нейрофотосессии

Небольшой лендинг-сайт для заказа стильной нейрофотосессии по вашему фото.

## Что внутри

- Лендинг-структура: блоки `преимущества`, `что вы получите`, `пакеты`, `заказ`, `подвал`
- Выбор пакета на карточках (кликабельные `Start / Pro / Brand`)
- Форма заказа с валидацией (имя, контакт, пакет)
- Поле для прикрепления фото (кликабельная “скрепка” + рекомендации)
- Фоллбек-картинок: если изображение отсутствует, показывается красивый плейсхолдер, чтобы верстка не “ломалась”
- Плавная прокрутка по якорям (`#packages`, `#order`, и т.д.)

## Как запустить локально

Вариант 1 (просто открыть): откройте `index.html` в браузере.

Вариант 2 (рекомендуется, чтобы корректно работали относительные пути): поднимите статический сервер из папки проекта.
Если у вас Python установлен, например:

```bash
python -m http.server 8080
```

Откройте: `http://localhost:8080`

## Изображения

Изображения должны лежать в папке `images/` и иметь имена из `images/README.txt`.

Если файлы не найдены — сработает плейсхолдер.

## Данные формы

Сейчас отправка формы имитируется фронтендом (без бэкенда).
Сообщение появляется в блоке `formMessage` и форма очищается.
